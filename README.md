# PowerBI_DACH_Transport_dashboard
DACH Region Transport Performance & Cost Audit

**Tech Stack:** SAP S/4HANA Data Simulation (SD/TM), Power Query, Power BI

## Business Context
The main goal of this report is the evaluation of delivery timeliness (OTIF - On-Time In-Full) from Poland to Germany and Austria, alongside a comparison of freight costs on specific routes to identify margin leakage.

## Data Architecture (SAP S/4HANA Simulation)
The dataset simulates standard structures from SAP modules:
* Sales and Distribution:** Billing documents
* Transportation Management:** Freight Orders and calculated costs
